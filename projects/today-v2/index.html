Layout Rules
Max width: 430px centered. Matches iPhone Pro viewport.
Page padding: space-xl (20px) horizontal. SectionCards use their own space-lg margin.
Scroll containers: Bleed to edges. Padding on items, not container.
Sticky bar: Always reserve env(safe-area-inset-bottom) for home indicator.
Interaction Patterns
Tap feedback: :active { opacity: 0.7 } on all interactive elements.
Transitions: all 0.15s ease on state changes.
Scroll snap: scroll-snap-type: x mandatory on horizontal scrolls.
Focus: Inputs get border-color: accent-blue. No other visible focus rings.
Toggle: JS class swap, one active at a time. No CSS-only toggles.
